# הוראות התקנה והפעלה למחשב חדש

קובץ זה מספק הוראות מלאות לריצת הבוט על מחשב חדש מגרסה שנשמרה ב-Git.

## 1. שכפול הפרויקט מהמחשב החדש

1. פתח PowerShell או Command Prompt.
2. שים שם את הקישור לריפו שלך והרץ:

   git clone <URL> telegram-bot
   cd telegram-bot

3. אם הפרויקט כבר נמצא במחשב החדש, עבור אל תיקיית הפרויקט.

## 2. התקנת Python

1. הורד והתקן Python 3.12 או גרסה עדכנית תואמת מ-https://www.python.org/downloads/
2. בזמן ההתקנה וודא שסימנת את האפשרות:
   - "Add Python to PATH"

## 3. יצירת סביבה וירטואלית

בחלון הטרמינל בתוך תיקיית הפרויקט הרץ:

   python -m venv .venv

לאחר מכן הפעל את הסביבה הווירטואלית:

   .\.venv\Scripts\activate

## 4. התקנת תלויות Python

לאחר הפעלת הסביבה הווירטואלית, הרץ:

   pip install --upgrade pip
   pip install -r requirements.txt

## 5. התקנת FFmpeg

הבוט משתמש ב-FFmpeg לעיבוד אודיו ווידאו.

1. הורד את גרסת Windows מ-https://ffmpeg.org/download.html
2. חילץ את התיקייה, למשל ל-`C:\ffmpeg\`
3. הוסף את `C:\ffmpeg\bin` ל-PATH של Windows.

אפשרות נוספת: אם רוצים להשתמש ב-FFmpeg מותקן במקום אחר, אפשר להגדיר את המשתנה:

   setx KARAOKE_FFMPEG_DIR "C:\path\to\ffmpeg"

ואז לפתוח מחדש את PowerShell.

## 6. קבלת טוקן לבוט טלגרם

1. פתח את הטלגרם וחפש את @BotFather
2. שלח את הפקודה `/newbot`
3. תן שם ו-username לבוט
4. שמור את הטוקן ש-BotFather מחזיר לך

## 7. הגדרת טוקן וסביבה

הבוט טוען את הטוקן מ-
- משתנה סביבה `TELEGRAM_BOT_TOKEN` או `BOT_TOKEN`, או
- קובץ `bot_token.txt`, או
- קובץ `.env` / `.env.local`

### אפשרות א: קובץ `bot_token.txt`

צור קובץ חדש בשם `bot_token.txt` בתיקיית הפרויקט והדבק בו רק את הטוקן, לדוגמה:

   123456789:ABCdefGHIjklMNOpqrSTUvwxyZ

### אפשרות ב: קובץ `.env`

צור קובץ `.env` בתיקיית הפרויקט עם התוכן:

   TELEGRAM_BOT_TOKEN=123456789:ABCdefGHIjklMNOpqrSTUvwxyZ

> קבצים אלו לא צריכים להיות ב-Git. ב-`.gitignore` כבר מוגדרים `bot_token.txt`, `.env`, `.env.local`.

## 8. הפעלת הבוט

בווידוא שהסביבה הווירטואלית פעילה והרצת הפקודות הקודמות, הרץ:

   python bot.py

אם הכל תקין, הבוט יתחיל לרוץ והמסך יישאר פתוח.

## 9. בדיקה ראשונית

1. פתח את הטלגרם
2. חפש את הבוט שלך
3. שלח `/start`
4. שלח קישור YouTube תקין

## 10. נקודות חשובות

- ודא ש-FFmpeg זמין ב-PATH או שהמשתנה `KARAOKE_FFMPEG_DIR` מצביע על תיקיית FFmpeg.
- ודא שסביבה וירטואלית הופעלה לפני התקנת התלויות והרצת `bot.py`.
- אם יש בעיות ריצה, בדוק את ההודעות בטרמינל.
- קבצים זמניים, לוקל סייט וקבצי לוג לא נשמרים ב-Git ויכולים להשתנות במחשבים אחרים.

## 11. שדרוג והפעלת סביבה חדשה מחדש

במחשב חדש לאחר עדכון הקוד מ-Git:

   git pull
   .\.venv\Scripts\activate
   pip install -r requirements.txt
   python bot.py

אם יש צורך, אפשר להוסיף משתני סביבה נוספים בקובץ `.env` לפי המסמכים של הפרויקט.

---

### הערה על העברה למחשב חדש

הקוד מגובה ב-Git, אך כדי שיתחבר וירוץ בצורה מלאה על מחשב חדש צריך גם:
- להתקין Python ו-FFmpeg
- להתקין את התלויות (`requirements.txt`)
- להגדיר טוקן טלגרם חוקי
- להגדיר בסביבה נתיבי FFmpeg במידה וצריך

בהצלחה!
