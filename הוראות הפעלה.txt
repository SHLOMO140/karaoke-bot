===================================================
  הוראות הגדרה והפעלה - בוט הקריוקי לטלגרם
===================================================

מה הבוט יודע לעשות:
- הורדת שירים מיוטיוב כ-MP3 או וידאו (בכל האיכויות)
- פלייבק קריוקי ללא ווקאל (הפרדת ווקאל עם Demucs)
- אקורדים + מילים (ניתוח מלודיה או שליפה מאתרי אקורדים)
- וידאו קריוקי עם כתוביות נצבעות בסנכרון מילה-מילה

שלב 1 - התקנות נדרשות:
--------------------------
1. Python 3.12 - https://www.python.org/downloads/
   (חשוב: סמן "Add Python to PATH" בזמן ההתקנה!)

2. ffmpeg - https://ffmpeg.org/download.html
   הבוט מאתר את ffmpeg אוטומטית לפי הסדר:
   - משתנה סביבה KARAOKE_FFMPEG_DIR (אם מוגדר)
   - ffmpeg שנמצא ב-PATH
   - התקנת winget קיימת או C:\ffmpeg\bin
   הכי פשוט: winget install Gyan.FFmpeg  (או חלץ ל-C:\ffmpeg והוסף ל-PATH)

3. Node.js - https://nodejs.org (נדרש ל-yt-dlp לעקיפת זיהוי בוטים של יוטיוב)

שלב 2 - יצירת בוט בטלגרם:
-----------------------------
1. פתח טלגרם וחפש @BotFather
2. שלח: /newbot
3. תן שם ו-username לבוט (חייב להסתיים ב-bot)
4. שמור את הטוקן שתקבל (נראה כך: 123456789:ABCdef...)

שלב 3 - הגדרת הטוקן:
-----------------------
צור קובץ בשם bot_token.txt בתיקיית הפרויקט והדבק בו את הטוקן בלבד.
(אפשרויות נוספות: משתנה סביבה TELEGRAM_BOT_TOKEN, או שורה
TELEGRAM_BOT_TOKEN=... בקובץ ‎.env)

מפתחות נוספים (אופציונלי, בקובץ ‎.env או ‎.env.local):
- GEMINI_API_KEY        - אימות מילים עם Gemini
- GOOGLE_API_KEY + GOOGLE_SEARCH_ENGINE_ID - חיפוש מילים בגוגל
- XAI_API_KEY           - ספק LLM חלופי (Grok)

שלב 4 - התקנה:
-----------------
הרץ פעם אחת את install.bat
(יוצר סביבה וירטואלית ב-.venv ומתקין את כל הספריות מ-requirements.txt)

שלב 5 - הפעלה ועצירה:
---------------------
- הפעלה: start_bot.bat   (רץ בלולאה עם הפעלה מחדש אוטומטית אחרי קריסה)
- עצירה:  stop_bot.bat

===================================================
שימוש:
===================================================
- שלח /start לבוט בטלגרם
- שלח קישור יוטיוב או טקסט לחיפוש
- בחר פורמט: MP3 / פלייבק קריוקי / וידאו / אקורדים / קריוקי וידאו
- בקריוקי וידאו: הבוט מתמלל, מאמת מילים ממקורות ברשת, ומבקש
  ממך לאשר או לתקן את הטקסט לפני יצירת הווידאו הסופי

===================================================
תחזוקה אוטומטית:
===================================================
- משימה מתוזמנת "KaraokeBot Daily Reset" רצה כל יום ב-04:00
  ומחזירה את סביבת העבודה לברירת מחדל: מוחקת פרויקטים ישנים,
  קבצים זמניים ולוגים. עבודות פעילות וביקורות פתוחות לא נמחקות.
- הרצה ידנית: reset_workspace.bat
  (אפשרויות: --dry-run להצגה בלבד, --force למחיקה גם של עבודות פעילות)
- סיכום כל ריצה נכתב ל-logs\cleanup.log

===================================================
הערות חשובות:
===================================================
- הבוט חייב לרוץ כל הזמן (אל תסגור את החלון של start_bot.bat)
- גודל קובץ מקסימלי לשליחה בטלגרם: ~45MB (וידאו גדול נדחס אוטומטית)
- עיבוד קריוקי מלא לוקח כמה דקות לשיר (תמלול + הפרדת ווקאל על CPU)
- עבודה כבדה אחת רצה בכל רגע; בקשות נוספות נכנסות לתור אוטומטית
- לוגים: logs\bot.log
===================================================
